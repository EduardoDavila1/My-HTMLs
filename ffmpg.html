FFmpeg Master Commands

Conversión de WebM a WebP conservando Alpha y Optimizando
Opción A: Calidad Balanceada (Recomendado) Standard
# Mantiene transparencia, loop infinito y reduce peso ffmpeg -i input.webm -c:v libwebp -lossless 0 -q:v 75 -loop 0 -an -vsync 0 output.webp
-lossless 0 Compresión con pérdida (necesaria para bajar peso).
-q:v 75 Calidad al 75%. (Bajar a 60 si pesa mucho).
-loop 0 Animación infinita.
-an Elimina el audio (ahorra bytes).
Opción B: Lossless (Calidad Perfecta) High Quality
# Sin pérdida visual. Archivo más pesado. ffmpeg -i input.webm -c:v libwebp -lossless 1 -loop 0 -an output_hd.webp
-lossless 1 Modo matemáticamente sin pérdida.
Troubleshooting: ¿Fondo negro? Fix
# Fuerza el formato de píxel RGBA para asegurar transparencia ffmpeg -i input.webm -vcodec libwebp -pix_fmt bgra -loop 0 -an output_fix.webp
-pix_fmt bgra Fuerza el canal Alpha explícito.
